# Important Keybinds

The most important keybind to know is **SUPER+K** — this opens a searchable menu with every keybind available.

SUPER is the key between Ctrl and Alt (Windows logo on most keyboards, Option on Mac).

## Essential Keybinds

| Keybind | Action |
|---------|--------|
| SUPER+SPACE | App launcher (also finds files and settings) |
| SUPER+Y | Shelly package installer |
| SUPER+SHIFT+Y | Unipkg TUI (works on non-Arch distros) |
| SUPER+R | Reload mangoWM config |
| SUPER+1-9 | Switch workspaces (hold SHIFT to move active window) |
| SUPER+Left Mouse | Drag and reorganize windows |
| SUPER+Right Mouse | Drag to resize windows |
| SUPER+Q | Close current window |
| SUPER+Z | Power menu (logout, reboot, shutdown) |
| SUPER+E | Open file browser |
| SUPER+B | Open Firefox |
| SUPER+ENTER | Open terminal |
| SUPER+H | Command help menu (SHIFT to open docs) |
| SUPER+X | Process manager / performance monitor |
| SUPER+K | Keybind reference menu |
