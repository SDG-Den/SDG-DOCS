# Included Software

SDG-OS comes with software on top of your base Linux install.

## From the Arch User Repository (AUR)

The AUR is a community-driven repository of packages for Arch Linux. Packages from the AUR are marked with `(aur)`.

- **mangowm-git** — the window manager (can be swapped for a pacman version)
- **dsearch-bin** — file search indexer for DankMaterialShell

## From the pacman repositories

These are official Arch Linux packages:

### Apps
- **Firefox** — web browser
- **Nautilus** — file browser
- **Satty** — screenshot editor
- **DMS Shell** — desktop shell environment
- **Shelly** — package management interface
- **Code** — code editor (opens project directories)

### Terminal
- **Ghostty** — terminal emulator
- **Zsh** — shell with plugins for history search, syntax highlighting, and autosuggestions
- **Powerlevel10k** — prompt theme
- **Stow** — symlink management utility
- **Zoxide** — smart directory jumper (replaces cd)
- **Eza** — modern ls replacement with icons and colors
- **Fzf** — fuzzy finder used across many tools
- **Micro** — terminal text editor

### Utilities
- **Brightnessctl** — screen brightness control
- **PipeWire** — audio and display capture
- **Yay** — AUR helper
- **Matugen** — automatic color theming
- **Adw-gtk-theme** — GTK theme support for auto-theming
- **Wl-clipboard** — clipboard interaction for scripts
- **Hyprshot / Grim** — screenshot tools
- **Kanshi** — monitor configuration
