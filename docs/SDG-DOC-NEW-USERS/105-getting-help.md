# Getting Help

## Documentation

| Tool | Usage | Description |
|------|-------|-------------|
| `sdgdocs` | `sdgdocs` | Browse all installed module docs |
| | `sdgdocs --from MODULE` | Skip module picker, go straight to a topic |
| `sdglinks` | `sdglinks` | Browse external reference links |
| `sdgtip` | `sdgtip` | Random tip |
| | `sdgtip list` | Browse all tips |
| | `sdgtip sources` | List which modules contribute tips |

## Keybinds

- `SUPER+K` — Searchable list of every active keybind
- `SUPER+H` — Command reference with tldr + man pages
- `SUPER+SHIFT+M` — Documentation browser (sdgdocs)
- `SUPER+SHIFT+/` — Config reference sheets
