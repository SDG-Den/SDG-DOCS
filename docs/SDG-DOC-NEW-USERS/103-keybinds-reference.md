# Keybinds Reference

The most important keybind category in SDG-OS is the `SUPER` key
(also known as the Windows key or Cmd key).

## Window Management

| Keybind | Action |
|---------|--------|
| `SUPER+Q` | Close active window |
| `SUPER+F` | Toggle fullscreen |
| `SUPER+T` | Toggle float/tile |
| `SUPER+P` | Pin window across workspaces |
| `SUPER+left/right/up/down` | Focus window in direction |
| `SUPER+Ctrl+left/right/up/down` | Swap panes |
| `SUPER+Alt+left/right/up/down` | Resize window |

## Workspaces

| Keybind | Action |
|---------|--------|
| `SUPER+1-9` | Switch to workspace N |
| `SUPER+SHIFT+1-9` | Move window to workspace N |

## System

| Keybind | Action |
|---------|--------|
| `SUPER+SPACE` | Application launcher |
| `SUPER+D` | Toggle dock |
| `SUPER+L` | Lock screen |
| `SUPER+R` | Reload configuration |
| `SUPER+SHIFT+Q` | Exit mangoWM |
| `SUPER+SHIFT+Y` | Package manager TUI |

## Help and Information

| Keybind | Action |
|---------|--------|
| `SUPER+H` | Command reference (tldr) |
| `SUPER+SHIFT+H` | Help/documentation browser |
| `SUPER+SHIFT+M` | Documentation browser (sdgdocs) |
| `SUPER+SHIFT+/` | Config reference sheets |
| `SUPER+K` | Keybinds list |
| `SUPER+M` | MangoWM config editor |
| `SUPER+0` | Random tip |
| `SUPER+X` | Process list |
| `SUPER+S` | Settings menu |
| `SUPER+W` | Wallpaper group selector |

## Media and Hardware

| Keybind | Action |
|---------|--------|
| `ALT+W` | Next wallpaper |
| `ALT+SHIFT+W` | Previous wallpaper |
| Volume keys | Volume control |
| Brightness keys | Brightness control |
