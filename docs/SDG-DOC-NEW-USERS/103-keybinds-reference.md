# Keybinds Reference

The most important keybind category in SDG-OS is the `SUPER` key
(also known as the Windows key or Cmd key).

Press `SUPER+K` at any time to see a searchable list of every active
keybind, filtered by your current keymode.

## Window Management

| Keybind | Action |
|---------|--------|
| `SUPER+Q` | Close active window |
| `SUPER+F` | Toggle fullscreen |
| `SUPER+ALT+F` | Toggle fake fullscreen (hides decorations) |
| `SUPER+T` | Toggle float/tile |
| `SUPER+P` | Pin window across workspaces |
| `SUPER+SHIFT+P` | Toggle overlay mode |
| `SUPER+left/right/up/down` | Focus window in direction |
| `SUPER+Ctrl+left/right/up/down` | Swap panes |
| `SUPER+Alt+left/right/up/down` | Resize window |
| `SUPER+SHIFT+left/right` | Focus previous/next in stack |
| `SUPER+SHIFT+up/down` | Move window up/down the stack |
| `SUPER+ALT+A` | Special layout toggle (varies by layout) |
| `SUPER+SHIFT+A` | Cycle through common layouts |

## Scratchpad

## Workspaces

| Keybind | Action |
|---------|--------|

| `SUPER+Alt+1-9` | Toggle workspace N visibility |
| `SUPER+J` | Mark current workspace for swap |
| `SUPER+SHIFT+J` | Swap with marked workspace |

## Keymodes

Press a function key to enter a context-sensitive keymode.  In each
mode, `WASD` controls are remapped.  Press `Escape` to return to
common mode.

| Key | Mode | WASD action |
|-----|------|-------------|
| `F1` | Window mode | Move windows |
| `F2` | Group mode | Group operations |
| `F3` | Scripts mode | (reserved) |
| `F4` | Theming mode | Wallpaper, bar presets, colors |
| `F6` | Settings | Opens the settings menu |
