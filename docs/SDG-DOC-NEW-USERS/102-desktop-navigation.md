# Desktop Navigation

## Workspaces

SDG-OS uses a dynamic tiling window manager with virtual workspaces.
You can switch between workspaces and move windows between them.

## The Bar

The top bar (managed by waybar through SDG-WAYSHELL-CONFIGS) shows:
- Active workspace indicators
- Clock
- System tray
- Volume/brightness controls
- Layout mode indicator

## App Launcher

Press `SUPER+SPACE` to open the application launcher.  Start typing
the name of the application and press Enter to launch it.

## Overview

Hold `ALT+TAB` to open the overview — an Exposé-style view of all
open windows across all workspaces.  Keep holding `ALT` and press
`TAB` to cycle, then release to focus the selected window.

## Notifications

Press `SUPER+SHIFT+N` to open the notification centre.  Notifications
from applications are collected here, with options to dismiss or
configure do-not-disturb.

## Quick Notes

Press `SUPER+N` to open the notepad — a simple scratchpad for quick
notes.
