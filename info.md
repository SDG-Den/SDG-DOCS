Package Name: sdg-docs
Descriptive Name: SDG Documentation Module
Source: https://github.com/SDG-Den/SDG-DOCS
Maintainer: SDGDen
Version:0.5.1

Dependencies: 
fzf, bat

Description: 
Official documentation module for SDG-OS. Provides an aggregated documentation browser (sdgdocs CLI) that lets users browse all markdown documentation from every installed SDG-OS module. Ships documentation suites covering Getting Started, New Users, Terminal, Tinkerers, Developers, AI Agents, and Troubleshooting, plus external reference links (sdglinks).
