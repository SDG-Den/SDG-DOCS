Package Name: sdg-docs
Descriptive Name: SDG Documentation Module
Source: https://github.com/SDG-Den/SDG-DOCS
Maintainer: SDGDen
Version:0.2

Dependencies: 
fzf, bat

Description: 
Official documentation module for SDG-OS. Provides an aggregated documentation browser (sdg-docs CLI) that lets users browse all markdown documentation from every installed SDG-OS module. Ships four documentation suites (New Users, Tinkerers, Developers, AI Agents) and external reference links.
